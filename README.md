## Getting Started
Follow the steps for setting up the project:
- git clone https://github.com/mukunda-p/CSC510-HW-42.git
- cd CSC510-HW-42
- pip install -r requirements.txt
    
## Unit Tests Execution
After setting up project follow the steps for unit tests execution:
- python -m pytest
- To add new test cases create new files under tests folder with name test_* or add the test cases in the exisiting test_math_methods.py
